## Required Evidence Before Advisor Reliance

- Corporate ownership and authorship records.
- Git history and release evidence reconciled to actual commits.
- Screenshots generated from the actual tool, not staged evidence.
- Monthly usage logs generated during real client delivery.
- Ticket references linked to usage IDs.
- Output-used and human-reviewed confirmation.
- Timesheets or work logs showing total productive SAP IS-U hours.
- Invoice/revenue mapping by client and service line.
- Anonymized incident examples and evidence hashes.
- Advisor-reviewed attribution methodology.
