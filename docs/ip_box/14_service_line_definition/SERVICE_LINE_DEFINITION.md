# Service Line Definition

## A. Standard SAP IS-U Consulting

Manual or minimally assisted consulting work. This may have low or no IP attribution.

## B. Software-Enabled SAP IS-U Technical Analysis

Work materially produced, accelerated, documented or quality-controlled through SAP IS-U Assistant. This is the main candidate service line for embedded IP income analysis.

Examples:

- Incident analysis using Standard and client KB.
- Debug checklists generated through the tool.
- Jira/email/client outputs materially assisted by the tool.
- Reusable resolution patterns captured in the tool.

## C. SAP IS-U Knowledge Engineering

Structuring, validating and improving reusable SAP IS-U knowledge inside the tool. This may support the IP asset where linked to development or exploitation.

## D. Admin / Commercial / Non-Qualifying Work

Excluded activities include invoicing, sales, negotiation, non-technical meetings, finance admin and unrelated consulting tasks.
