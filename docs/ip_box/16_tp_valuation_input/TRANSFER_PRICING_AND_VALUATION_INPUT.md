# Transfer Pricing and Valuation Input

This is not a transfer pricing study. It is an input document for advisors.

## IP Description

SAP IS-U Assistant is a proprietary internal software asset used to support SAP IS-U consulting delivery.

## Service Line

The candidate service line is Software-Enabled SAP IS-U Technical Analysis.

## Functions Performed by the Software

- Knowledge retrieval.
- Incident structuring.
- Source-aware AI output generation.
- Standard/client namespace handling.
- Evidence and dossier generation.
- Usage logging and monthly reporting.

## Functions Performed by the Consultant

- Client communication.
- SAP system access and validation.
- Professional judgment.
- Final decision-making.
- Manual review of generated outputs.

## Assets Used

- Proprietary source code.
- Standard and client KB structures.
- Qdrant vector index.
- OpenAI API.
- Local SQLite repositories.

## Risks Assumed

- Development risk.
- Quality and accuracy risk.
- Confidentiality risk.
- Client delivery risk.
- Commercialization risk if external licensing is pursued.

## Possible Valuation Methods for Advisors

- Contribution analysis.
- Residual profit split.
- Relief from royalty.
- Cost-plus for routine manual work.

High attribution may be arguable only if the tool is central to delivery and supported by real evidence.
