# Roadmap and Commercialisation Options

## Phase 1: Internal Desktop Tool

Current phase. The tool supports internal SAP IS-U consulting delivery.

## Phase 2: Mature Internal Workflow

Integrate usage logging into UI, enforce operating procedure and collect several months of real evidence.

## Phase 3: Internal Reporting and IP Evidence

Generate monthly reports, ticket-level evidence packs, revenue mapping and advisor packs.

## Phase 4: Optional Private SaaS

Potential future model for controlled external users. Requires security, licensing, privacy and SAP IP review.

## Phase 5: Optional Desktop Licence

Potential packaging for individual SAP IS-U consultants or internal enterprise teams.

## Phase 6: Optional Managed SAP IS-U Support Platform

Potential managed support service where the software is central to delivery.

## Phase 7: Optional Partnerships

Potential partnership model with SAP consultancies. Requires commercial and IP boundary review.

Current exploitation is internal. No external commercialization is claimed as current fact.
